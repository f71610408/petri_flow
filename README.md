## Installation
Add this line to your application's Gemfile:

```ruby
gem 'petri_flow', require: 'wf'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install petri_flow
```

Install graphviz

```
brew install graphviz
``

Migration:

```
bundle exec rake wf:install:migrations
```
